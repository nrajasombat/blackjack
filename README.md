#Basic black jack game in react/redux using my playing card lib


Prerequisites
-------------
```node 9.4.0```

```npm 5.6.0```


Instructions to run
-------------

Install [nvm](https://github.com/creationix/nvm) if you haven't already

```nvm install v9.4.0```

```nvm use```

Start server by:

```npm start```



Production
----------
```npm run build```
